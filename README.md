# AWS Resource Tracker (Bash + Cron)

## Overview
This project automates AWS resource monitoring using a Bash script and Cron job.

## Features
- Lists S3 Buckets
- Lists EC2 Instances
- Lists Lambda Functions
- Lists IAM Users
- Automated execution using Cron (every 5 minutes)

## Tech Stack
- AWS CLI
- Linux (Ubuntu)
- Bash scripting
- Cron
